When running VPP commands, always remember:

1. Do not run any destructive command.
2. Do not run any command which deletes anything.
3. If it is necessary to delete or remove something, always ask for permission.
4. All the VPP commands would be present in reference.md. You can also use generic VPP commands if they are not present in this file.
